Holds classes for handling data from a gps device (serial, network, file, etc.) and provides some applications to work with this data. The main application is {@link org.dinopolis.gpstool.Gpsylon} - a moving map application.

A simple command line demo is {@link org.dinopolis.gpstool.GPSTool} - retrieve the position from a gps device, and upload/download tracks/waypoints/routes from/to a garmin gps device.
