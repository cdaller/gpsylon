This is a GPSylon plugin that allows the user to open a frame that
displays the memory usage of the GPSylon application.

To install the plugin, copy the jar file and the lib directory to the
plugin directory of GPSylon. Start GPSylon and the "Plugins" menu shows
an entry "Memory Monitor".

This plugin uses the MemoryMonitor from the Batik project.

For the latest version of GPSylon and this plugin, please visit
http://gpsmap.sourceforge.net.
